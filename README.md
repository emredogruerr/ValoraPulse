ValoraPulse — Dynamic Price Tracking Demo

ValoraPulse is a simple yet clean Flask application that simulates price changes for different products and visualizes their price history.
The goal of the project is to track how a product’s price fluctuates over time and display it with a smooth, modern-looking chart.

Features

Simulated real-time price fluctuations

SQLite database for storing price history

Automatic price logging every time a product page is opened

Clean and responsive UI

Smooth matplotlib chart with a custom dark theme

Tech Stack

Python / Flask

SQLite

SQLAlchemy

Matplotlib

Bootstrap 5

How it works

When you visit any product page, the app generates a new price point and appends it to the product’s history.
The chart updates automatically, giving a quick overview of recent price movement.
